# Platinummonkey Tap

## How do I install these formulae?

`brew install platinummonkey/tap/<formula>`

Or `brew tap platinummonkey/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "platinummonkey/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
